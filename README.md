 # TaskMaster

TaskMaster is a full-stack task management application built to help teams efficiently create,
assign, prioritize, and track tasks across departments.

## Tech Stack
- Frontend: Next.js
- Backend: ASP.NET Core Web API
- Authentication: JWT
- Database: PostgreSQL
- Containerization: Docker & Docker Compose

## Features
- User authentication with role-based access control
- Task creation, assignment, and prioritization
- Task status tracking and comments
- Email notifications (optional)
- Responsive UI for desktop and mobile

## Project Structure
/frontend  -> Next.js application  
/backend   -> ASP.NET Core API
